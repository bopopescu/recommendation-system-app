# recommendation-system-app
 A recommender app built with Python and Angular 8 using a MySQL DB.

## Installation

#### Angular CLI

Download Nodejs -> [https://nodejs.org/en/]
It installs NPM as well.


Angular installation
```
npm install -g @angular/cli
ng new my-dream-app
cd my-dream-app
ng serve
```
